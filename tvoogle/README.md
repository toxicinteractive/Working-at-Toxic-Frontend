# Tvoogle

## To run this project:

Navigate to this folder using the terminal - the Tvoogle folder.

# 1 - Install dependencies

Run the following command in the terminal.

### `npm install`

# 2 - Run the project

Run the following command in the terminal.

### `npm start`

## Comments
